# Datasets

Datasets live here as usable assets. Formal dataset papers belong in Zotero.

## Index Rules

- Record dataset name, task, modality, embodiment, license when available, paper link, code/data link, and source repository.
- Use `type: dataset` in [[90_Sources/Resource Registry]].
- Add to Zotero when the dataset has a formal paper.

## Source Streams

- HCPLab-SYSU for broad dataset coverage.
- zchoi for VLA and robotics datasets.
- ChanganVR for navigation and embodied vision datasets.
- Awesome-Touch for tactile and visual-tactile datasets.
